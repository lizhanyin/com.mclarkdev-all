# com.mclarkdev-all
com.mclarkdev LibAPIs 系列库的整合

由于原维护者不再向中央仓库推送新版本，我希望自行拉取并编译这些库。但这些库分散在多个仓库，拉取和编译较为麻烦，因此创建了这个整合库。

整个库的实际代码版权归属 [mclarkdev.com](https://github.com/LibAPIs)。

本库仅用于整合，如果有人认为这侵犯了版权，请在 issue 中留言，我将删除此库。

包含：

[liblog](https://github.com/LibAPIs/LibLog.git)

[libloggelf](https://github.com/LibAPIs/LibLogGELF.git)

[libwebsvc](https://github.com/LibAPIs/LibWebSvc.git)

[libobjectcache](https://github.com/LibAPIs/LibObjectCache.git)

[libmetrics](https://github.com/LibAPIs/LibMetrics.git)

[libdb](https://github.com/LibAPIs/LibDB.git)

[libsockets](https://github.com/LibAPIs/LibSockets.git)

[libapriltag](https://github.com/LibAPIs/LibAprilTag.git)

[libusbkey](https://github.com/LibAPIs/LibUsbKey.git)

[libtmachine](https://github.com/LibAPIs/LibTMachine.git)

[libkeyhook](https://github.com/LibAPIs/LibKeyhook.git)

[libargs](https://github.com/LibAPIs/LibArgs.git)

[libextras](https://github.com/LibAPIs/LibExtras.git)

[libobjectpooler](https://github.com/LibAPIs/LibObjectPooler.git)

[liblightify](https://github.com/LibAPIs/LibLightify.git)
